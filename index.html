java基础语法

Posted on 2018-07-02

关键字：赋予了特殊含义的单词
标识符：数字，字母，$,_数字不可以开头
首字母大写是规范
编译器：翻译器
注释：
作用：1对程序进行说明2对程序进行调试
分类：//单行注释
/多行注释/
/*只有java才有的注释——文本注释/——javadoc可以提取这个注释，形成html文档，并对程序进行说明
变量类型11种：
基本数据类型：byte,short,int,long,float,double,char,boolean
引用数据类型：class,interface,数组
整数默认类型：int
小数默认类型：double
自动类型提升，从低字节准转换为高字节的大小。
byte a=3和a=a+1;的区别，3是常量，a+1是变量，不确定。
a=a+1;//编译器会报错
a+=1;//不会报错，因为在这个式子中，自动实现了变量的类型转换
编译时期保证语法正确。
由ASCII编码表——发明了键盘这个设备，键盘的电路板保存字符和二进制码的转换。
JB2312，JBK——中国的编码
char有2个字节，范围0——65535
位运算符：与或非，异或
逻辑运算符：与或非，&&，||，!
单&和双&&的区别：
单&两边都参与运算，当左边为false时，右边仍然进行操作
左移运算符：空位补0，往左移几位乘以2的几次幂。移位运算符只能操作2的倍数
右移运算符须得区分符号位：

有符号右移，高位补原位

为无符号右移，高位补0，
对一个数异或2次不变，可以实现简单的加密

java基础常识

Posted on 2018-07-01

软件：一系列按照特定顺序组织的计算机数据和指令的集合。
系统软件：windows，DOS，linux
应用软件：qq，迅雷
人机交互的方式：图形化界面（GUI）和命令行界面(CLI)，先有命令行后有图形化界面
windows中的命令：help dir cd\
计算机语言：人与计算机交流的方式
java来自SUN1995推出来的一门高级编程语言，Oracle为数据库，Java
面向对象贼鸡儿难
Java语言的三种技术框架：EE,SE,ME（学SE,企业用EE）
java语言的特点：
跨平台：java可以在不同的操作系统上跑
为啥：因为windows中没有解释java的东西，oracle公司推出了jvm（核心）应用软件来跑他的java程序，java程序能在各个平台上运行，都是基于java虚拟机上的东西
JRE:java运行环境=jvm+核心类库
JDK:java开发工具包=开发工具+JRE
环境变量的设置：set只能在当前的命令窗口生效
java只能在当前文件夹中编译运行。
使用javac来运行java的源码，自动生成demo的字节码（类文件，Java程序文件）——调用java的编译器，对Java源码进行翻译。
使用java来运行这些类文件
java命令区分大小写，执行java时候，必须省略.class后缀
不是所有的java程序都需要主方法，需要根据需求来定义。主方法是用来保证程序的运行。
class Demo 通过class关键字定义了一个名称为Demo的类
public static void main(String[]) 定义了一个主方法，固定
System.out.println(“你好，hello”); 输出语句，作用是将括号中的数据显示在屏幕上
class Demo{
public static void main(String[] args)
{
System.out.println(“你好，hello”);
}
}
常见问题：
1找不到java文件
路径名错误/文件名错误/默认的文件名错误
2找不到或者无法加载主类
jvm没有找到要运行的类
3没有保存导致无法显示你修改后的值
4代码中含有中文
5jvm，如果你配置了虚拟机的classpath，则会在你配置的classpath找类，如果没有，会在当前目录下找
